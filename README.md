## Description
This is a CMS style blog where users can create accounts, posts, as well as see other posts and comment on them. This project uses javascript and handlebars to generate the HTML, and bootstrap was used for styling.

## Usage
Use npm i to install dependencies.

## Testing
To run the program, create a .env with login credentials to mysql, source the schema, seed the database, and run npm start into the console.
